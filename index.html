What we do

Our project is Element database and Analytical system.

Project Requirements: Designing the system, and implementing a database and analytical tools (including functionalities for search, visualization, basic statistical analysis, and classification tasks) for foodstuffs.

Project Specifications: 1.Data. Data collection cleaning and standardization will be performed to provide the content of the database.

2.Database and analysis system. The software product will have the following elements:
a.Software (user interface, database, analysis system, help pages, and visualization module)
b.Testing and maintenance components (these will be basic for a prototype)
